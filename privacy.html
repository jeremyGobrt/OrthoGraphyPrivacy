Politique de Confidentialité

Correcteur de texte ChatGPT

Dernière mise à jour : 25/09/2024

Votre vie privée est importante pour nous. Cette politique de confidentialité explique quelles informations sont collectées par l'extension "Correcteur de texte ChatGPT", comment nous utilisons ces informations et comment nous les protégeons.

1. Informations que nous collectons

Clé API OpenAI : Lorsque vous utilisez l'extension, vous avez la possibilité de saisir une clé API OpenAI. Cette clé est stockée localement dans votre navigateur, via l'API `chrome.storage.local`, et est utilisée uniquement pour faire des requêtes à l'API ChatGPT. Elle n'est jamais collectée par nous ou partagée avec des tiers.

Texte sélectionné : L'extension traite uniquement le texte que vous sélectionnez pour la correction. Ce texte est envoyé à l'API OpenAI dans le cadre de la correction, mais n'est jamais collecté ou stocké par nous.

2. Comment nous utilisons les informations

Clé API OpenAI : La clé API est utilisée uniquement pour authentifier les requêtes à l'API ChatGPT afin de corriger le texte sélectionné. Nous n'utilisons pas cette clé pour d'autres finalités.

Texte sélectionné : Le texte que vous sélectionnez est temporairement envoyé à l'API OpenAI pour correction et est retourné à l'extension. Nous ne stockons, ne partageons, ni ne collectons ce texte.

3. Partage des informations

Nous ne partageons aucune donnée personnelle avec des tiers. Les informations (clé API et texte sélectionné) sont uniquement transmises à l'API OpenAI pour permettre la correction du texte.

4. Sécurité des informations

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger la clé API et le texte sélectionné. Ces données sont stockées uniquement localement dans votre navigateur et ne sont jamais transmises à des serveurs autres que celui d'OpenAI pour le traitement de la correction.

5. Modifications de cette politique

Nous nous réservons le droit de mettre à jour cette politique de confidentialité à tout moment. Nous vous informerons de toute modification en publiant la nouvelle politique sur cette page.
